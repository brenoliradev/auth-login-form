# Auth Logins

This is a POC made for study authentication on forms.

* Already using:
  * React.js / Node.js 
  * [Login/Register API;](https://github.com/lirbre/node-js-jwt-auth-mongodb)
  * Yup, Formik - to verify form;
  * Material UI and SASS - to stylize;
  * React-router-dom; 
  * Axios.
